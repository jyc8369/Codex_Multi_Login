# Codex Multi Login

This is a VS Code extension for people who use more than one Codex account.
It helps you keep accounts in one place and switch between them more easily.

## What it does

- Add a Codex account
- Switch the active account
- Check quota again
- Save account information
- Open a simple dashboard

## Easy use

1. Install the extension.
2. Open the command palette and run `Codex Multi Login: Open Dashboard`.
3. Add an account if needed.
4. Switch to the account you want.
5. Check quota when needed.

## Notes

- Some features only work in the desktop version of VS Code.
- The extension keeps account data in VS Code storage.
- More details are in the developer guide.

<a name="korean"></a>
# Codex Multi Login

VS Code에서 Codex 계정을 여러 개 사용하는 사람을 위한 확장입니다.
계정을 한 곳에 모아 두고 더 쉽게 전환할 수 있게 도와줍니다.

## 할 수 있는 일

- Codex 계정 추가
- 활성 계정 전환
- quota 다시 확인
- 계정 정보 저장
- 간단한 대시보드 열기

## 사용 방법

1. 확장을 설치합니다.
2. 명령 팔레트를 열고 `Codex Multi Login: Open Dashboard`를 실행합니다.
3. 필요하면 계정을 추가합니다.
4. 원하는 계정으로 전환합니다.
5. 필요할 때 quota를 확인합니다.

## 참고

- 일부 기능은 VS Code 데스크톱 버전에서만 동작합니다.
- 확장은 계정 데이터를 VS Code 저장소에 보관합니다.
- 자세한 설명은 개발자 문서를 보세요.
